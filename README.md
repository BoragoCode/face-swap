换脸程序
====

# 安装
```bash
git clone https://github.com/JeffTrain/face-swap
cd face-swap
pip install -r requirements
```

# 使用
```bash
python face-swap.py ted_cruz.jpg donald_trump.jpg
```

# 效果
以上示例命令会把特德·克鲁斯的脸部换到唐纳德·特朗普的照片上。
